API Example - Node.js

hello.js


var http = require('http');
http.createServer(function (req, res) {
  res.writeHead(200, {'Content-Type': 'text/plain'});
  res.end('Hello World\n');
}).listen(1337, '127.0.0.1');
console.log('Server running at http://127.0.0.1:1337/');

Command line:


$ node hello.js
Server running at http://127.0.0.1:1337/

Data Formats

  • Spreadsheets
    • Human interface. Bad for APIs and automation.
  • CSV (comma-separated-value) Files
    • Better, but support only simple schemata and require writing custom parsers.
  • XML
    • Supports complex schemata, but still requires custom parsers.
  • JSON
    • Supports complex schemata without custom parsers.

JSON Schema Standards

  • JSON-LD
    • JSON-LD W3C Specification
    • Dataset Schema at schema.org
  • JSON API
    • Very new. Simpler and less ambitious than JSON-LD.
